# Study Agent 项目规则

## 项目概述
这是一个**智能学习助手 Agent**，基于智谱 GLM-4-Flash 模型，通过 OpenAI SDK 调用，提供学习任务管理、艾宾浩斯遗忘曲线复习计划、学习考察等功能。

## 技术栈
- **语言**: Python 3
- **LLM**: 智谱 GLM-4-Flash（通过 OpenAI SDK 调用）
- **依赖**: openai, python-dotenv, schedule, tqdm, requests, pydantic
- **运行环境**: WSL Ubuntu，Python 虚拟环境 (.venv)
- **启动方式**: bash start_agent.sh 或直接 python3 agent.py

## 项目结构
- agent.py — 全部代码（969行），包含以下核心类：
  - MemoryStore — 长期记忆：JSON 文件持久化（study_memory.json），管理任务和学习日志
  - ChatSession — 短期记忆：对话历史 + 滑动窗口截断（默认5轮）
  - StudyAgent — 核心逻辑：LLM 调用 + Function Calling（工具：add_task, update_task, record_daily, get_task_ranking, mark_reviewed）
  - App — 交互层：菜单驱动 + 自然语言输入 + 定时调度
- requirements.txt — 依赖清单
- start_agent.sh — 后台启动脚本
- .env — API Key 配置（不提交到 Git）

## 核心功能
1. **添加学习任务** — 任务名、截止日期、分类（编程/算法/数学/英语/其他）、掌握程度（生疏/一般/熟悉/精通）、预估时长
2. **生成复习计划** — 基于遗忘曲线 + 优先级排序，复盘昨日学习 + 制定今日计划
3. **记录学习内容** — 按日期记录每日学习内容
4. **任务优先级排行** — 综合截止紧急度 + 掌握程度 + 复习间隔计算优先级分数
5. **标记已复习** — 更新复习次数和最后复习时间
6. **学习考察模式** — 基于遗忘曲线智能出题，答后自动更新掌握程度
7. **自由对话** — 支持自然语言交互，通过 Function Calling 调用工具
8. **定时提醒** — 每天 08:00 和 21:00 自动触发复习计划生成

## 优先级计算公式
优先级 = 截止紧急度(10-40) + 掌握程度分(0-30) + 复习间隔分(0-30)

## 开发约定
- 所有代码在 agent.py 单文件中，新增功能在此文件中扩展，在后续的阶段可以进行分文件
- 数据持久化使用 JSON 文件（study_memory.json）
- LLM 调用使用流式输出（stream=True）
- 新增工具需在 StudyAgent.TOOLS 中定义 schema，并在 _execute_tool 中实现
- 中文界面，中文注释
- 不需要生成文件,只需要把修改的文件完整输出
- 每次修改的时候都要说明修改的内容，包括新增的代码、修改的代码、删除的代码等为什么这样改
