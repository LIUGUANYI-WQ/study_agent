## 1. 项目概述

### 1.1 项目简介

本项目是一个**个人学习提醒 Agent**，基于大语言模型（LLM）实现智能学习辅助功能。

### 1.2 主要功能

| 功能编号 | 功能名称 | 功能描述 |
| :--- | :--- | :--- |
| 1 | 添加学习任务 | 支持设置截止日期、分类、掌握程度 |
| 2 | 生成复习计划 | 基于艾宾浩斯遗忘曲线自动生成 |
| 3 | 记录学习内容 | 支持结构化记录和多轮优化 |
| 4 | 任务优先级排行 | 根据截止时间、掌握程度计算优先级 |
| 5 | 学习考察模式 | 智能出题考察学习成果 |
| 6 | 自由对话 | 支持自然语言交互 |

## 3. 核心功能实现详解

### 3.1 结构化学习记录

**实现目标**：将学习内容存储为结构化数据，便于后续知识图谱构建。

**代码位置**：`agent.py:165-186`

**核心数据结构**：

```python
structured_record = {
    "raw_content": str,           # 原始学习内容
    "knowledge_points": list,     # 知识点列表
    "mastery_level": str,         # 掌握程度
    "tags": list,                 # 标签列表
    "summary": str,               # 核心要点总结
    "quality_score": int,         # 质量评分(0-100)
    "create_time": str            # 创建时间
}

# 知识点结构
knowledge_point = {
    "name": str,                  # 知识点名称
    "type": str,                  # 类型：概念/原理/方法/工具/案例/其他
    "description": str            # 详细描述
}
```

### 3.2 质量评分机制

**实现目标**：自动评估学习记录质量，过滤低质量噪音数据。

**代码位置**：`agent.py:188-210`

**评分规则**：

| 评分维度 | 分值范围 | 评分标准 |
| :--- | :--- | :--- |
| 内容长度 | 0-30分 | ≥50字得30分，≥20字得15分 |
| 知识点数量 | 0-40分 | ≥3个得40分，≥1个得20分 |
| 知识点详细度 | 0-30分 | 每个详细描述加10分，最高30分 |
| **总分** | **0-100分** | 三项相加取最小值 |

### 3.3 LLM内容分析

**实现目标**：使用大语言模型分析学习内容，提取关键信息。

**代码位置**：`agent.py:414-461`

**分析流程**：

1. **质量评估**：1-10分评分
2. **知识点提取**：提取3-5个核心知识点
3. **要点总结**：生成不超过100字的总结
4. **改进建议**：针对低质量内容提供优化建议
5. **标签建议**：生成3-5个标签

### 3.4 多轮对话优化

**实现目标**：通过交互式对话优化学习记录质量。

**代码位置**：`agent.py:463-553`

**对话流程**：

```
┌─────────────────────────────────────────────────────────────┐
│                    多轮对话优化流程                          │
├─────────────────────────────────────────────────────────────┤
│  第一轮：用户输入学习内容                                    │
│         ↓                                                  │
│  LLM分析内容质量、提取知识点、生成总结                        │
│         ↓                                                  │
│  判断质量评分是否 < 7分                                     │
│         ├─ 是 → 显示改进建议 → 询问是否优化 → 用户修改       │
│         │          ↓                                        │
│         │   重新分析优化后的内容                            │
│         ↓                                                  │
│  第二轮：确认用户掌握程度                                    │
│         ↓                                                  │
│  第三轮：用户确认是否保存                                    │
│         ↓                                                  │
│  保存结构化学习记录                                         │
└─────────────────────────────────────────────────────────────┘
```

## 4. 代码结构解析

### 4.1 类结构关系

```
┌────────────────────────────────────────────────────────┐
│                      App                              │
│  (应用入口，管理菜单交互和定时任务)                      │
│        │                                              │
│        ▼                                              │
│  ┌──────────────┐                                     │
│  │  StudyAgent  │ ◄────────────────────────────┐      │
│  │  (核心逻辑)   │                             │      │
│  └──────┬───────┘                             │      │
│         │                                     │      │
│         ├─────────────────┬───────────────────┤      │
│         ▼                 ▼                   ▼      │
│  ┌───────────┐   ┌─────────────┐   ┌─────────────┐   │
│  │MemoryStore│   │ ChatSession │   │   OpenAI    │   │
│  │(长期记忆)  │   │ (短期记忆)   │   │   Client    │   │
│  └───────────┘   └─────────────┘   └─────────────┘   │
└────────────────────────────────────────────────────────┘
```

### 4.2 关键方法速查表

| 类名 | 方法名 | 功能描述 |
| :--- | :--- | :--- |
| MemoryStore | `record_daily_structured()` | 存储结构化学习记录 |
| MemoryStore | `_calculate_quality_score()` | 计算学习记录质量分数 |
| MemoryStore | `get_all_knowledge_points()` | 提取所有知识点 |
| StudyAgent | `analyze_learning_content()` | 使用LLM分析学习内容 |
| StudyAgent | `optimize_learning_record()` | 多轮对话优化记录 |
| App | `_handle_record()` | 处理记录学习内容请求 |

### 4.3 数据流向图

```
用户输入 → App._handle_record()
              ↓
        StudyAgent.optimize_learning_record()
              ↓
        StudyAgent.analyze_learning_content()
              ↓
        OpenAI API (glm-4-flash)
              ↓
        解析LLM响应 → 提取知识点、评分、建议
              ↓
        用户确认优化 → 循环改进
              ↓
        MemoryStore.record_daily_structured()
              ↓
        保存到 study_memory.json
```
